# 基于Spring Boot + Vue停车场管理系统 java158

## 项目概述


## 技术栈

- **后端**：Spring Boot、MyBatis、MySQL
- **前端**：Vue、Element Plus
- **数据分析**：ECharts
- **安全认证**：JWT
- **地图服务**：高德API
- **通知服务**：邮箱通知

## 功能模块

- **用户管理**：管理系统的用户账户。
- **停车场管理**：监控和管理停车场信息。
- **车辆管理**：登记和管理车辆信息。
- **停车记录**：记录和管理车辆的停车历史。
- **缴费管理**：处理和管理停车费用。
- **公告管理**：发布和管理通知公告。
- **数据分析**：分析停车数据，提供决策支持。
- **地图服务**：提供停车场位置信息和导航。
- **通知服务**：发送邮件提醒。

## 系统角色

- 系统管理员
- 停车场工作人员
- 普通用户

## 运行环境

- **后端**：Java 1.8+、MySQL 5.7+
- **前端**：现代浏览器

## 部署步骤

1. 配置后端环境，包括数据库。
2. 编译前端资源。
3. 部署前后端代码到服务器。
4. 启动后端服务。
5. 访问前端界面。
